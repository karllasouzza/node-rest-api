# RF

[ ] o usuário deve poder criar uma nova transação;
[ ] o usuário deve poder obter o resumo da conta;
[ ] o usuário debe poder listar as transações realizadas;
[ ] o usuário deve poder visualizar uma transação única;

# RN

[ ] A transação pode ser do tipo crédito que somará o valor ao saldo ou do tipo débito que subtrairá o valor do saldo;
[ ] deve ser identificar o usuário entre as requisições;
